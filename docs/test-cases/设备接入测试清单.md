# 设备接入测试清单

## 一、USB 识别

- 能识别 VID/PID
- 能识别 interface/class
- 能判断是否支持 ADB
- 能判断是否支持 RNDIS
- 能判断是否支持串口

## 二、连接测试

- ADB 是否可连接
- RNDIS 是否可发现 IP
- SSH/Telnet 是否可探测
- 串口是否可打开

## 三、配置读取测试

- 能读取关键配置文件
- 能提取终端 ID
- 能提取车牌
- 能提取平台服务器
- 能提取白名单

## 四、日志读取测试

- 能读取业务日志
- 能读取启动日志
- 能导出日志片段

## 五、诊断包测试

- 能导出诊断包
- 诊断包包含配置、日志、设备能力、连接方式
