# 视频接入规划

## 一、目标

后期在 APP 中接入设备视频能力，包括实时预览、抓图、录像导出、通道查看。

## 二、视频能力分级

### V1：信息级
- 查看设备是否支持视频
- 查看视频通道数量
- 查看主码流/子码流信息

### V2：预览级
- 连接设备实时预览
- 选择通道
- 选择码流

### V3：控制级
- 截图
- 录像片段导出
- 音频开关
- 通道映射查看

## 三、与现有架构的结合

视频能力不单独做孤立模块，而作为统一工作台的一个 feature：

- feature-video-center
- profile 中声明 videoAvailable
- connector 中声明 videoTransport 能力

## 四、注意事项

- 视频接入可能走 HTTP / RTSP / RTP / JT1078 / 私有协议
- 先做能力抽象，再做具体协议适配
