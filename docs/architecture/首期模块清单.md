# 首期模块清单

## 一、目标

明确第一阶段真正要创建的模块，避免一开始铺得太大，影响交付速度。

## 二、首期必须创建的模块

### 1. app
职责：

- 启动应用
- 导航
- 承载工作台

### 2. core:common
职责：

- Result
- Error
- Logger
- Dispatcher
- 基础模型

### 3. core:usb
职责：

- USB 枚举
- USB descriptor 分析
- 权限处理

### 4. core:network
职责：

- IP 探测
- 端口扫描
- 网络连通性检查

### 5. core:terminal
职责：

- 终端会话抽象
- 文本流/字节流统一模型

### 6. core:file
职责：

- 文件读取抽象
- 目录浏览抽象

### 7. core:parser
职责：

- ini 解析
- log 解析
- sqlite 信息提取

### 8. core:device
职责：

- 设备指纹
- 设备能力模型
- 设备摘要

### 9. connectors:connector-adb
职责：

- ADB 探测
- ADB 会话
- shell / file / diagnostics

### 10. connectors:connector-rndis
职责：

- USB 网卡识别
- RNDIS IP 探测
- 常见端口服务发现

### 11. connectors:connector-serial
职责：

- 串口设备识别
- 串口收发会话

### 12. profiles:profile-generic-linux
职责：

- 通用 Linux 路径与规则

### 13. profiles:profile-jt808-ai-recorder
职责：

- JT808 + AI + 记录仪通用画像

### 14. features:feature-device-discovery
职责：

- 设备发现页
- 能力识别页
- 推荐连接方式页

### 15. features:feature-terminal
职责：

- 统一终端页

### 16. features:feature-config-center
职责：

- 统一配置中心页

### 17. features:feature-log-center
职责：

- 统一日志页

### 18. features:feature-diagnostics
职责：

- 诊断信息汇总
- 导出诊断包

## 三、首期暂缓模块

- connector-ssh

- connector-rawusb
