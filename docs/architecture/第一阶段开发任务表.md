# 第一阶段开发任务表

## 目标

完成可落地 MVP：识别设备、选择连接方式、进入终端、读取关键配置。

## 任务拆分

### 1. 项目骨架
- 创建多模块结构
- 建立基础依赖管理
- 建立公共结果模型

### 2. USB 核心
- USB 枚举
- 权限申请
- interface/class 识别

### 3. 连接器
- ADB Connector
- Serial Connector
- RNDIS Connector

### 4. 设备画像
- Generic Linux
- Generic JT808
- Rockchip Recorder

### 5. 功能页
- 设备发现页
- 终端页
- 配置中心页
- 日志中心页

### 6. 首批读取能力
- 终端 ID
- 车牌号
- 服务器地址与端口
- 白名单
- JT808 参数区
