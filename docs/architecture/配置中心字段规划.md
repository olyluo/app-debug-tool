# 配置中心字段规划

## 一、目标

把当前已知设备的共性参数抽象成统一字段组，便于后续多设备复用。

## 二、建议字段分组

### 1. 设备身份
- terminalId
- serialNumber
- licensePlateNo
- licensePlateColor
- vehicleType
- vin

### 2. JT808 平台
- mainServerAddr1
- mainServerPort1
- backServerAddr1
- backServerPort1
- mainServerAddr2
- mainServerPort2
- mainServerAddr3
- mainServerPort3
- registerId
- authCode
- cityStandard
- protocolVersion

### 3. 网络参数
- apn
- apnUser
- apnPassword
- wifiHotspotState
- rndisIp

### 4. 远程控制与短信
- smsNumber
- smsAlarmNumber
- monitorTel
- resetTel
- rebuildTel
- listenTel

### 5. 白名单与视频
- avWhitelist
- videoChannelCount
- mainStreamInfo
- subStreamInfo

### 6. AI/报警参数
- adasEnable
- dmsEnable
- bsdEnable
- alarmPolicy
- capturePolicy
- thresholdConfig

## 三、落地原则

- 先定义统一字段名
- 再由 profile 提供设备字段到统一字段的映射
- UI 层只面向统一字段，不直接依赖某个 ini 键名
