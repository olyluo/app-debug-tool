# 首期里程碑

## 里程碑 M1：项目可启动

目标：

- 建立 Android 多模块工程骨架
- 建立 app / core / connectors / profiles / features 基本结构
- 建立基础依赖注入与导航

完成标志：

- 工程可以正常打开
- 模块可以正常编译
- 首页可以启动

## 里程碑 M2：设备可识别

目标：

- 完成 USB 枚举
- 完成设备能力初步判断
- 能识别 ADB / RNDIS / Serial

完成标志：

- 插入设备后能显示识别结果
- 能展示推荐连接方式

## 里程碑 M3：设备可进入

目标：

- 完成 ADB Connector
- 完成 RNDIS Connector
- 完成 Serial Connector

完成标志：

- 至少一种通道可进入设备
- 可以打开统一终端页

## 里程碑 M4：设备可读懂

目标：

- 完成 Generic Linux Profile
- 完成 Generic JT808 Profile
- 完成首个 Rockchip/JT808 记录仪画像

完成标志：

- 能提取终端 ID
- 能提取车牌
- 能提取服务器参数
- 能提取白名单或远程参数区

## 里程碑 M5：设备可诊断

目标：

- 完成日志中心
- 完成配置中心
- 完成诊断包导出

完成标志：

- 能导出基础诊断结果
- 能支撑新设备首轮接入分析
