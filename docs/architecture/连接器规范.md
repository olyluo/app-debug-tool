# 连接器规范

## 一、目标

所有连接方式必须实现统一接口，避免上层页面和业务逻辑直接依赖某个具体协议。

## 二、标准连接器类型

- AdbConnector
- SerialConnector
- RndisConnector
- SshConnector
- HttpConnector
- RawUsbConnector

## 三、统一能力接口

每个连接器应至少支持以下能力中的一部分：

- probe()
- connect()
- disconnect()
- openTerminal()
- runCommand()
- listFiles()
- readFile()
- writeFile()
- collectDiagnostics()
- getConnectionInfo()

## 四、连接优先级建议

默认优先级：

1. ADB
2. RNDIS
3. SSH/Telnet
4. Serial
5. HTTP
6. Raw USB

## 五、连接器输出要求

连接器在连接成功后，应输出统一的设备会话信息：

- connectionType
- sessionId
- remoteAddress
- shellAvailable
- fileReadAvailable
- fileWriteAvailable
- logReadAvailable
- videoAvailable
