# 设备画像规范

## 一、为什么需要设备画像

不同设备虽然都基于 Linux + AI + JT808，但配置路径、日志目录、数据库位置、业务进程名、特殊能力不同。

设备画像负责描述这一差异。

## 二、设备画像应描述的内容

- 设备匹配规则
- 常见配置文件路径
- 常见日志路径
- 常见数据库路径
- 关键参数提取规则
- 健康检查规则
- 推荐连接器
- 快捷动作
- 视频能力定义

## 三、基础画像分层

### 1. GenericLinuxProfile
适用于所有 Linux 类设备。

### 2. GenericJt808Profile
适用于带 JT808 的设备。

### 3. VendorSpecificProfile
适用于特定厂商或特定设备族。

## 四、推荐字段

- profileId
- displayName
- matchRules
- configLocations
- logLocations
- dbLocations
- extractors
- healthChecks
- quickActions
- supportedFeatures
