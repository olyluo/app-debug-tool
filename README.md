# APP调试工具

一个面向 Linux + AI + JT808 类设备的结构化安卓调试平台。

项目目标：

- 统一接入多种底层 Linux 设备
- 通过 USB / ADB / RNDIS / 串口 / SSH 等方式进入设备
- 自动识别设备能力并推荐最佳连接方式
- 统一查看配置、日志、终端、JT808 参数、白名单、视频能力
- 支持后期扩展视频接入、诊断包导出、脚本动作编排

核心设计原则：

- 不按单一设备开发，按能力层开发
- 连接方式插件化
- 设备画像可扩展
- UI 按任务组织，不按协议组织
- 所有关键动作可沉淀为可复用脚本动作

推荐技术方向：

- Android App
- Kotlin
- Jetpack Compose
- 多模块架构
- 协程 + Flow
- 插件化 Connector / Profile 扩展

首期范围：

- USB 设备识别
- ADB 接入
- RNDIS 网络探测
- 串口接入
- 统一终端页
- 配置中心（终端 ID / 车牌 / 服务器 / 白名单）
- 日志中心
- 诊断包导出

后期范围：

- 视频接入与预览
- 设备录像/截图导出
- JT808 参数映射中心
- 动作脚本中心
- 厂家私有 USB 协议支持
- 多设备画像市场/模板
